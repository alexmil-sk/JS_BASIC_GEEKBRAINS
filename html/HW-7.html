Решение

1. Выводить счёт в режиме реального времени.

Пояснения

1. В верстке HTML создаем заголовок над таблицей : "h3 class="score-string">Вывод текущего счета/h3"
2. Создаем глобальную переменную: var currentScore.
3. В функции function prepareGameField() обращаемся к созданному тэгу и присваиваем ему класс:
currentScore = document.querySelector('h3');
currentScore.classList.add('score-string');
4. В функции "function haveFood(unit)" создаем строку: "currentScore.innerHTML = 'Ваш текущий счет: ' + score;"

2. Генерировать временные препятствия на поле.

Пояснения

1. Создаем css-свойство для класса препятствий: "obstacle-unit".
2. Cоздаем функцию "createObstacles()" для формирования ячеек-препятствий и присвоения им класса css-свойств.
3. Создаем функцию "haveObstacle(obst)" для контроля прохождения через препятствие.
4. В функции "startGame()" создаем интервал запуска функции createObstacles() создания препятствий:
"setTimeout(createObstacles, 7000);".
5. В функции "move()" в строку проверки добавляем проверку на препятствие:
"if (!isSnakeUnit(new_unit) && new_unit !== undefined && !haveObstacle(new_unit)) {}".

3. *Убрать границы поля, т.е. при пересечении границы поля змейка появляется с противоположной стороны.

Пояснения

В функции "move()" указываем дополнительные условия для ячеек типа "undefined" и для координат по направлениям движения змейки.
