Домашние задания по курсу JavaScript (Базовый)

ДЗ-1

1.Задать температуру в градусах по Цельсию. Вывести в alert соответствующую температуру в градусах по Фаренгейту.

Подсказка.

Pасчёт идёт по формуле:

Tf = (9 / 5) * Tc + 32,

где Tf – температура по Фаренгейту,

Tc – температура по Цельсию.

2.Объявить две переменные: admin и name. Записать в name строку "Василий"; Скопировать значение из name в admin. Вывести admin (должно вывести «Василий»).

3. *Чему будет равно JS-выражение 1000 + "108"?

4.*Самостоятельно разобраться с атрибутами тега script ('async' и 'defer'). Прокомментировать полученную информацию.

5.Выполнить обмен значений между двумя переменными без использования третьей переменной. (А = 10; B = 20;)
